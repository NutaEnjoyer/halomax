# OpenAI Realtime API - Доступные голоса

## Все 8 голосов поддерживают:
- ✅ Русский язык
- ✅ Узбекский язык
- ✅ Таджикский язык
- ✅ Английский и другие языки
- ✅ Естественные интонации
- ✅ Низкая латентность (real-time)

## Список голосов:

### 1. **Alloy** (по умолчанию)
- **Характеристика**: Neutral, balanced
- **Описание**: Нейтральный, сбалансированный голос
- **Подходит для**: Универсальное использование, профессиональная коммуникация
- **Тон**: Средний, спокойный

### 2. **Ash**
- **Характеристика**: Clear, articulate
- **Описание**: Четкий, артикулированный голос
- **Подходит для**: Технические объяснения, инструкции
- **Тон**: Ясный, профессиональный

### 3. **Ballad**
- **Характеристика**: Smooth, calm
- **Описание**: Плавный, спокойный голос
- **Подходит для**: Медитации, расслабляющий контент
- **Тон**: Мягкий, успокаивающий

### 4. **Coral**
- **Характеристика**: Warm, friendly
- **Описание**: Теплый, дружелюбный голос
- **Подходит для**: Продажи, customer service, приветствия
- **Тон**: Дружественный, располагающий

### 5. **Echo**
- **Характеристика**: Deep, resonant
- **Описание**: Глубокий, резонирующий голос
- **Подходит для**: Серьезные объявления, авторитетные сообщения
- **Тон**: Низкий, уверенный

### 6. **Sage**
- **Характеристика**: Wise, measured
- **Описание**: Мудрый, размеренный голос
- **Подходит для**: Консультации, обучение, советы
- **Тон**: Взвешенный, опытный

### 7. **Shimmer**
- **Характеристика**: Bright, energetic
- **Описание**: Яркий, энергичный голос
- **Подходит для**: Промо, реклама, мотивация
- **Тон**: Высокий, бодрый

### 8. **Verse**
- **Характеристика**: Expressive, dynamic
- **Описание**: Выразительный, динамичный голос
- **Подходит для**: Сторителлинг, презентации
- **Тон**: Эмоциональный, живой

## Рекомендации по выбору:

### Для продаж и поддержки клиентов:
- **Coral** - теплый и дружелюбный
- **Alloy** - профессиональный и нейтральный

### Для технической поддержки:
- **Ash** - четкий и артикулированный
- **Sage** - мудрый и размеренный

### Для маркетинга и промо:
- **Shimmer** - энергичный и яркий
- **Verse** - выразительный и динамичный

### Для серьезных сообщений:
- **Echo** - глубокий и авторитетный
- **Sage** - взвешенный и опытный

### Для расслабляющего контента:
- **Ballad** - плавный и спокойный

## Как используется в HALO AI:

1. **Frontend**: Пользователь выбирает голос из выпадающего списка
2. **Backend**: Передает выбор в Voximplant через customData
3. **Voximplant**: Настраивает OpenAI Realtime API с выбранным голосом
4. **OpenAI**: Генерирует речь с выбранным голосом на нужном языке

## Технические детали:

```javascript
// В Voximplant сценарии:
realtimeClient.sessionUpdate({
    session: {
        audio: {
            output: {
                voice: "coral" // Выбранный голос
            },
            input: {
                transcription: {
                    model: "whisper-1",
                    language: "ru" // Язык транскрипции
                }
            }
        }
    }
});
```
